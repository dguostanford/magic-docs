---
title: "Get Processor Token"
api: "POST https://api-dev.join-magic.com/api/merchants/payment_preprocess/get_processor_token/"
description: "This Api has three steps performed internally:
        1. Get processor token from Plaid for Dwolla
        2. Add funding source on Dwolla for Customer Bank account
        3. Initiate A2A transfer and store Transfer information in Checkout object
"
---

</ParamField>

### Response

<ResponseField name="success" type="number">

</ResponseField>
