---
title: "Get Checkout Object"
api: "GET https://api-dev.join-magic.com/api/merchants/checkout"
description: "This endpoint used for get Checkout objects."
---

### header
<ParamField header="api-key" type="string">
  Merchant API key.
</ParamField>

### Response

<ResponseField name="success" type="number">
  This will return 200 on success.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api-dev.join-magic.com/api/merchants/checkout' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>'
```

</RequestExample>
